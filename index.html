Meme what you say


Meme what you say is the final presentation piece from the Northcoders bootcamp. I, along with my amazing team-mates, built a real-time multiplayer iOS game in the space of about 10 days. We are extremely proud of it!


The idea of the game is to come together with friends or family (3 - 8 players) and vote on each others witty captions for the randomly generated memes. Funniest player wins!


The back end of the app was made using FireBase which provided the cloud FireStore database and the Cloud Storage where we were able to keep the photos. The front end was built using React Native and Expo.

NC News


NC News is a "reddit style" web page that allows users to vote on, post or delete articles and comments.


For the back end I built an API using PSQL and then interacted with it using Knex. This is then hosted on Heroku in order to be accessed by the front end.

The front end is then built using React, styled with CSS and the hosted on Netlify.


You will see on the homepage that there is a login section. In order to successfully login you will need to grab one of the existing publishers usernames (look at articles and see who posted them), and also the password. This, just for eases sake, is always "password". Once logged in you should be able to see a my articles section and you should also gain the ability to delete comments made by your user.
