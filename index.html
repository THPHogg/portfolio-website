Meme what you say


Meme what you say is the final presentation piece from the Northcoders bootcamp. I, along with my amazing team-mates, built a real-time multiplayer iOS game in the space of about 10 days using React Native, FireStore, FireBase and a lot of styling. We are extremely proud of it!


The idea of the game is to come together with friends or family (3 - 8 players) and vote on each others witty captions for the randomly generated memes. Funniest player wins!


Please click on the title to see a short presentation video showing off the app, how we made it, why we used the technologies that we did and any obstacles we faced during this process.


Or alternatively click here to go to the GitHub repo to have a look at the files.

NC News


NC News is a "reddit style" web page that allows users to vote on, post or delete articles and comments.


For the back end I built an API using PSQL and then interacted with it using Knex. This is then hosted on Heroku in order to be accessed by the front end.

The front end is then built using React, styled with CSS and the hosted on Netlify.
